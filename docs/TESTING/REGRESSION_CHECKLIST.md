# Checklist de Regressão

Antes de qualquer merge ou release, valide:

- [ ] `npm run build` executa com sucesso.
- [ ] `npm run test:unit` não apresenta falhas.
- [ ] Não existem erros de lint (`npm run lint`).
- [ ] Runner carrega sem erros no console.
- [ ] Runner mantém estado de vitória/derrota consistente.
- [ ] Documentação atualizada (docs relevantes).
