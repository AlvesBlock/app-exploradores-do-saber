# Checklist de Regressao

Antes de qualquer merge ou release, valide:

- [ ] `npm run build` executa com sucesso.
- [ ] `npm run test:unit` nao apresenta falhas.
- [ ] Nao existem erros de lint (`npm run lint`).
- [ ] Hub carrega com modulos principais e card `Plus`.
- [ ] `Galeria Encantada` permanece bloqueada ate a conclusao dos 4 modulos.
- [ ] `Galeria Encantada` persiste progresso apos recarregar a pagina.
- [ ] Runner carrega sem erros no console.
- [ ] Runner mantem estado de vitoria e derrota consistente.
- [ ] Documentacao atualizada.
