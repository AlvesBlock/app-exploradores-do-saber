# UI/UX Mobile: EcoNave

## Principios

- portrait mobile como layout prioritario
- CTAs grandes e proximos ao polegar
- HUD curta e legivel
- overlays com alto contraste e scroll quando necessario
- stage frame com proporcao vertical fixa

## Controles

- direcional em dock inferior
- botao grande de disparo
- botao separado para pulso
- teclado opcional para desktop

## Feedbacks

- bubble curta com icone para acerto, erro e power-up
- boss bar so aparece quando necessario
- cards de resultado priorizam metricas e aprendizado

## Integracao nativa com o app

- usa `PrimeVue Button`, `Dialog` e `Toast` como shell
- preserva linguagem visual infantil do app
- o card do Hub segue o mesmo padrao dos extras ja existentes
