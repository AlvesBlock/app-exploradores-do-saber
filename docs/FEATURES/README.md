# Diretrizes de Features

Este diretório contém modelos e registros para implementar novas funcionalidades no projeto.

## Fluxo de Feature
1. Criar uma especificação usando `TEMPLATE_FEATURE_SPEC.md`.
2. Implementar em pequenas etapas.
3. Registrar progresso e decisões em `TEMPLATE_IMPLEMENTATION_LOG.md`.
4. Atualizar documentação e changelog.
